== Description ==

WP Carousel is a plugin that create a carousel with a posts, category's posts or pages, and you can put it anywhere on the blog.

In the plugin's options page you can add items to each carousel and there you can set up some options and the theme used in each carousel.

The plugin uses jQuery and StepCarousel, and load it directly from your WordPress installation files, so if you have a WordPress version without jQuery, it won't work.

== Installation ==

1. Upload `wp-carousel` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php wp_carousel(CAROUSEL'S ID); ?>` in your templates

More info [here](http://sumolari.com/plugins-temas/wp-carousel/)
